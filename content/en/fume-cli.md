Install the CLI using `yarn` or `npm`

<code-group>
  <code-block label="Yarn" active>

  ```bash
  yarn global add fume-cli
  ```

  </code-block>
  <code-block label="NPM">

  ```bash
  npm install -g fume-cli
  ```

  </code-block>
</code-group>
